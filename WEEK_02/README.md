## 백준 10988 (팰린드롬인지 확인하기)  
:pushpin: 유형
* 배열

:question: 문제  
* 문자열이 앞뒤가 같은지 확인

:heavy_check_mark: 해결
* 입력받은 문자열의 반까지만 반복 + end를 문자열의 마지막 인덱스로 설정
* 처음과 end의 문자를 비교해서 다르면 flag를 0으로 변경(팰린드롬 X) + break
* 같으면 end 인덱스를 1 감소시킴

## 백준 8958 (OX퀴즈)
:pushpin: 유형
* 조건, 분기

:question: 문제
* OX로 이루어진 문자열의 점수 구하기
* O가 연속될 경우 그 문제의 점수는 연속된 O의 개수가 됨

:heavy_check_mark: 해결

* X가 나오면 누적 O 점수를 1로 리셋해줌
